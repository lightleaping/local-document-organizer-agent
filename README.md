# Local Document Organizer Agent

PDF, TXT, Markdown 문서를 업로드하면 **대표 주제, 핵심 키워드, 추천 파일명, 요약, 중복 의심 문서, Markdown 리포트**를 생성하는 로컬 문서 정리 자동화 도구입니다.

핵심 키워드, 대표 주제, 추천 파일명은 같은 분석 기준을 공유하도록 설계했습니다. 따라서 추천 파일명은 단순히 단어를 이어붙인 결과가 아니라, 문서의 대표 주제와 상위 키워드를 조합한 결과입니다.

---

## 주요 기능

| 기능 | 설명 |
|---|---|
| 문서별 카드 UI | 표 가로 스크롤 없이 문서별 결과를 카드로 표시 |
| 핵심 키워드 | 복합 표현 우선 추출, 조사와 불용어 제거, 중요도 순 표시 |
| 대표 주제 | 핵심 키워드 기반 문서 유형 판단 |
| 추천 파일명 | 대표 주제와 상위 핵심 키워드를 조합 |
| 요약 | 문서 내부의 핵심 문장을 추출 |
| 중복 탐지 | 유사 문서를 중복 후보로 표시 |
| 리포트 | Markdown 리포트 미리보기 및 다운로드 |

---

## 실행

```bash
python -m streamlit run app.py
```

```bash
python main.py --input sample_docs --output reports --summary-sentences 3 --keywords 6 --duplicate-threshold 0.65
```

---

## 출력 예시

```text
[1] sample_ai.txt
    대표 주제     : AI 문서
    핵심 키워드   : 인공지능, 자연어 처리, 이미지 인식, 추천 시스템, 문서 자동화
    추천 파일명   : ai__인공지능_자연어_처리.txt
    요약          : 인공지능은 데이터를 기반으로 패턴을 학습하고 예측이나 분류를 수행하는 기술이다.
```

## 포트폴리오 포인트

- CLI와 Streamlit 웹 UI를 모두 제공
- 사용자 친화적인 문서별 카드형 결과 화면 구현
- 키워드, 대표 주제, 추천 파일명을 하나의 분석 흐름으로 연결
- 외부 LLM API 없이 로컬 문서 처리 파이프라인 구현
- RAG 전처리 과정으로 확장 가능한 구조
