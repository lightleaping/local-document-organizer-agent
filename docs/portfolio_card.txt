Local Document Organizer Agent

PDF, TXT, Markdown 문서를 업로드하면 문서 유형, 제목, 핵심 키워드, 추천 파일명, 요약, 중복 후보, Markdown 리포트를 생성하는 로컬 문서 정리 자동화 도구입니다.

Python, Streamlit, PyMuPDF, scikit-learn을 활용해 외부 LLM API 없이 동작하는 문서 처리 파이프라인을 구현했습니다. 단순 키워드 나열 방식이 아니라 문서 유형을 먼저 판정하고, 유형에 맞는 제목·핵심 키워드·추천 파일명·요약을 생성하도록 개선했습니다.

주요 기능
- PDF/TXT/Markdown 문서 파싱
- 문서 유형 판정
- 유형별 제목 생성
- 핵심 키워드 선별
- 유형 기반 추천 파일명 생성
- 문장형 요약 생성
- 문자 n-gram 기반 중복 문서 탐지
- Markdown 분석 리포트 자동 생성
- Streamlit 웹 UI 제공

기술 스택
Python, Streamlit, PyMuPDF, scikit-learn, TF-IDF, cosine similarity, argparse, Markdown

포트폴리오 포인트
- 문서 기반 AI 서비스의 전처리 파이프라인 구현
- 외부 API 없는 로컬 자동화 도구 설계
- CLI와 웹 UI를 모두 제공하는 실무형 도구 구성
- 사용자 원본 파일을 직접 변경하지 않는 dry-run 방식 적용
- 추천 파일명과 요약 결과의 일관성 개선
