Local Document Organizer Agent

PDF, TXT, Markdown 문서를 업로드하면 문서별 요약, 키워드 추출, 파일명 추천, 중복 탐지, Markdown 리포트 생성을 수행하는 로컬 문서 정리 자동화 도구입니다.

Python, PyMuPDF, scikit-learn TF-IDF, cosine similarity를 활용해 외부 LLM API 없이 동작하는 문서 분석 파이프라인을 구현했습니다. CLI뿐 아니라 Streamlit 웹 UI를 추가하여 문서 업로드, 분석 결과 표, 중복 후보 확인, 리포트 미리보기와 다운로드까지 사용자 친화적으로 구성했습니다.

주요 기능
- PDF/TXT/Markdown 문서 파싱
- 문서별 추출식 요약
- TF-IDF 기반 키워드 추출
- 키워드 기반 추천 파일명 생성
- 문자 n-gram 기반 중복 문서 탐지
- Markdown 분석 리포트 자동 생성
- Streamlit 웹 UI 제공

기술 스택
Python, Streamlit, PyMuPDF, scikit-learn, TF-IDF, cosine similarity, argparse, Markdown

포트폴리오 포인트
- 문서 기반 AI 서비스의 전처리 파이프라인 구현
- 외부 API 없는 로컬 자동화 도구 설계
- CLI와 웹 UI를 모두 제공하는 실무형 도구 구성
- 사용자 원본 파일을 직접 변경하지 않는 dry-run 방식 적용
