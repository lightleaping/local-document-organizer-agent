- 단순 키워드 나열이 아니라 문서 유형을 먼저 판정합니다.

- 핵심 키워드는 문서 유형을 설명하는 주요 개념만 선별합니다.
- 요약은 사용자가 문서 내용을 빠르게 이해할 수 있도록 문장형으로 정리합니다.
