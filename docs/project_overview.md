# Project Overview

## 프로젝트명

Local Document Organizer Agent

## 한 줄 설명

PDF, TXT, Markdown 문서를 업로드하거나 로컬 폴더로 지정하면 요약, 키워드 추출, 파일명 추천, 중복 탐지, Markdown 리포트 생성을 수행하는 문서 정리 자동화 도구입니다.

## 개발 목적

문서 기반 업무에서는 여러 파일을 직접 열어 핵심 내용을 확인하고, 비슷한 파일을 구분하고, 파일명을 정리하는 반복 작업이 자주 발생합니다.

본 프로젝트는 이러한 반복 작업을 자동화하기 위해 개발되었습니다. 특히 RAG 시스템이나 문서 기반 AI 서비스에서 필요한 문서 수집, 텍스트 추출, 전처리, 태깅, 중복 제거 흐름을 작은 로컬 도구로 구현하는 데 초점을 두었습니다.

## 주요 기능

- PDF, TXT, Markdown 문서 텍스트 추출
- 문서별 추출식 요약
- TF-IDF 기반 키워드 추출
- 키워드 기반 추천 파일명 생성
- 문자 n-gram 기반 중복 문서 탐지
- Markdown 리포트 생성
- Streamlit 웹 UI
- CLI 옵션 실행

## 사용자 흐름

1. 사용자가 웹 UI에서 문서를 업로드하거나 CLI에서 폴더를 지정합니다.
2. 시스템이 문서 텍스트를 추출합니다.
3. 문서별 요약과 키워드를 생성합니다.
4. 키워드를 바탕으로 추천 파일명을 생성합니다.
5. 문서 간 유사도를 계산해 중복 의심 문서를 표시합니다.
6. 분석 결과를 표와 Markdown 리포트로 제공합니다.

## 포트폴리오 의미

이 프로젝트는 문서 기반 AI 서비스의 전처리 파이프라인을 이해하고, 이를 사용자 친화적인 도구로 구현할 수 있음을 보여줍니다.
